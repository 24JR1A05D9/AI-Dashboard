# 🎬 Netflix AI Dashboard

## 📌 Overview
This project is an interactive dashboard built using Streamlit to analyze Netflix dataset.

## 🚀 Features
- Key Metrics (KPIs)
- Data Cleaning
- Interactive Filters
- 5+ Visualizations
- Movies vs TV Shows analysis
- Country-wise insights
- Ratings distribution

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## ▶️ Run Project
```bash
streamlit run app.py
